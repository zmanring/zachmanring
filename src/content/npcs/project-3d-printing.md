When the market doesn't make the part you need, you design it and print it yourself.

I've been using 3D modeling and FDM printing to fill gaps — weird, specific, niche problems that no manufacturer addresses because the market is too small to bother.

Things I've designed, tested in the field, and now sell:
→ Radio mounts for the 2026 Maverick X3 XRC center console
→ TYT TH8600 bezel for the Maverick X3 XRC
→ Ram mount magnetic mic holder
→ Seadoo Switch rail cover mount with hook
→ RV ceiling antenna / satellite dish cap
→ Festool hose clamps and CNC clamp riser blocks

Each one started the same way: "this should exist, but doesn't." Model it in CAD, print a prototype, test it in the real environment, iterate until it fits. Same feedback loop as software — different material.

The models are available on the Southern Ginger Workshop site for other makers who hit the same wall.
