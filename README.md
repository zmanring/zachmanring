## Tech Stack

- **Phaser 3** — game engine
- **React 18** — UI layer (HUD, info cards, directory, mobile view)
- **Vite + TypeScript** — build tooling
- **Custom event bus** — bridges Phaser ↔ React via `window` CustomEvents
- **Procedural sprites** — all characters and world elements drawn in code
